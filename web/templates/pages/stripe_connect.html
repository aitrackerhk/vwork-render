透過 Stripe Connect 安全收款，不需要管理 API Key

  • 支援信用卡、Apple Pay、Google Pay
  • 不需要自己管理 API Key
  • 自動轉帳到你的銀行帳戶
  • 可在 Stripe Express Dashboard 查看交易記錄
開始前請準備以下資料：
  1. 銀行帳號（用於接收付款）
  2. 身份驗證文件（身份證或護照）
註冊過程約 5-10 分鐘，需在 Stripe 頁面上完成驗證並同意服務條款。

使用說明
如何啟用 Stripe Connect 收款？
  1. 點擊下方「開始連接 Stripe」按鈕
  2. 系統會將你導向 Stripe 頁面，依指示完成以下步驟：
    • 同意 Stripe 服務條款
    • 填寫銀行帳號（接收付款用）
    • 驗證身份資料
  3. 完成後 Stripe 會自動跳回此頁面，狀態會顯示「已連接」
  4. 前往付款方式管理新增一個「Stripe Connect」類型的付款方式
  5. 將該付款方式設為「網店付款方式」即可在網店使用
費用說明：Stripe 會收取每筆交易的手續費（一般為 2.9% + HK$2.35）。 平台會另外收取每筆交易金額的 2% 作為服務費。詳情請參考 Stripe 的定價頁面。
