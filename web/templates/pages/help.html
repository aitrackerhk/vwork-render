商品管理

新增商品
  1. 前往 產品管理 → 產品
  2. 點擊 新增產品 按鈕
  3. 填寫產品基本信息：名稱、描述、價格
  4. 設定庫存數量（如果啟用庫存管理）
  5. 選擇產品分類、品牌、屬性等
  6. 上傳產品圖片
  7. 設定產品條碼或二維碼（可選）
  8. 儲存產品
搜尋、篩選及排序商品

在產品列表頁面，您可以：

  • 使用搜尋框搜尋產品名稱或條碼
  • 使用篩選器按分類、品牌、價格範圍等篩選
  • 點擊欄位標題進行排序（名稱、價格、庫存等）
  • 使用進階搜尋功能進行多條件搜尋
匯入及匯出商品

批量管理商品：

  • 匯出商品：點擊 匯出 Excel 或 匯出 PDF 按鈕
  • 匯入商品：點擊 匯入 按鈕，選擇 Excel 檔案，系統會自動解析並匯入商品資料
  • 匯入前請確保 Excel 檔案格式正確，包含必要的欄位（名稱、價格等）
商品條碼及二維碼
  • 系統支援自動生成商品條碼和二維碼
  • 在 POS 系統中，可以使用掃描器掃描條碼快速添加商品
  • 可以列印條碼標籤貼在商品上
{{ else if eq .Category "import-export" }}

匯入 / 匯出

您可以在多個模組（例如：客戶、商品、訂單、服務單、採購單、發票、庫存報表等）的列表頁面，使用 匯出 或 匯入 Excel 來批量處理資料。

匯出（Excel / PDF）
  1. 進入任一資料列表頁（例如：客戶、商品、訂單）
  2. 在列表右上角的操作區，點擊 匯出 Excel 或 匯出 PDF
  3. 系統會下載檔案到您的電腦

提示：若列表沒有資料，系統會提示「沒有資料，不能匯出」。

匯入（Excel）
  1. 進入支援匯入的列表頁
  2. 點擊 匯入 Excel（或先下載 匯入模板）
  3. 依照模板欄位填寫後上傳 Excel，系統會自動解析並匯入

提示：若看到「下載匯入 Excel（不含資料/含資料）」表示該模組提供模板功能；匯入前請先確認欄位格式與必填欄位。

訂單管理

創建訂單
  1. 前往 訂單管理 → 訂單
  2. 點擊 新增訂單 按鈕
  3. 選擇客戶（或創建新客戶）
  4. 添加產品或服務項目：
    • 點擊 添加產品 按鈕
    • 搜尋並選擇產品
    • 設定數量和單價
    • 如有需要，設定折扣
  5. 選擇付款方式（現金、信用卡、轉帳等）
  6. 選擇運送方式（如適用）
  7. 填寫備註（可選）
  8. 點擊 儲存 完成訂單創建

訂單狀態包括：

  • 報價單：報價單狀態的訂單
  • 已確認：已確認的訂單
  • 處理中：正在處理的訂單
  • 已完成：已完成的訂單
  • 已取消：已取消的訂單

您可以在訂單詳情頁面更改訂單狀態。

發票生成
  1. 在訂單詳情頁面，點擊 生成發票 按鈕
  2. 系統會自動生成發票號碼
  3. 可以預覽發票內容
  4. 可以列印或下載 PDF 格式的發票
報價單管理

創建報價單：

  1. 前往 訂單管理 → 報價單
  2. 點擊 新增報價單
  3. 填寫報價單信息（類似創建訂單）
  4. 報價單可以轉換為正式訂單

POS 自助

什麼是 POS 自助？

POS 自助（Kiosk Mode）是專為零售門店設計的自助收銀模式，讓顧客可以自行完成結帳流程，無需店員協助。

使用 POS 自助
  1. 前往 POS 收銀台
  2. 點擊右上角的 Kiosk 模式 按鈕
  3. 系統會切換到自助收銀界面
  4. 顧客可以自行掃描商品條碼或搜尋商品
  5. 添加商品到購物車
  6. 選擇付款方式完成結帳
POS 自助功能特點
  • 簡潔界面：隱藏不必要的功能，專注於核心結帳流程
  • 快速結帳：優化操作流程，提升結帳效率
  • 多種付款：支持現金、信用卡、移動支付等多種付款方式
  • 會員登入：顧客可以登入會員帳號，享受會員優惠
  • 自助服務：適合無人店、自助服務區等場景
自助服務設定
  • 在 設定 → POS 設定 中配置自助模式選項
  • 設定自助模式的預設付款方式
  • 設定是否允許顧客修改商品數量
  • 設定是否顯示商品詳細信息

庫存管理

庫存自動更新

系統會在以下情況自動更新庫存：

  • 創建訂單時，自動減少庫存
  • 創建採購單時，自動增加庫存
  • 取消訂單時，自動恢復庫存
出入貨記錄
  1. 前往 庫存管理 → 出入貨記錄
  2. 查看所有庫存變動記錄
  3. 可以按日期、產品、倉庫等篩選
  4. 可以匯出記錄報表
庫存盤點
  1. 前往 庫存管理 → 庫存盤點
  2. 點擊 新增盤點
  3. 選擇倉庫
  4. 輸入實際庫存數量
  5. 系統會自動計算盤點差異
  6. 確認後更新庫存
低庫存預警
  • 系統會自動監控庫存數量
  • 當庫存低於設定值時，會在 庫存管理 → 低庫存預警 中顯示
  • 可以設定每個產品的最低庫存數量

動態字段功能

什麼是動態字段？

動態字段功能允許您為任何數據表添加自定義字段，無需修改數據庫結構。這讓您可以根據業務需求靈活擴展系統功能。

如何使用動態字段？
  1. 進入任何數據管理頁面（如客戶、產品、訂單等）
  2. 點擊 編輯 按鈕
  3. 在表單中找到 動態字段 區域
  4. 點擊 新增字段 按鈕
  5. 輸入字段名稱和值
  6. 選擇字段類型（文字、數字、日期等）
  7. 儲存後，這些自定義字段會顯示在該記錄中
動態字段的應用場景
  • 為客戶添加特殊標記或備註
  • 為產品添加額外的規格信息
  • 為訂單添加特殊要求
  • 儲存業務特定的額外資訊

vAi 是系統內建的 AI 智能助手，可以幫助您處理各種業務問題，回答系統使用問題，協助數據分析，提供業務建議等。

  • 最新項目
  • 近期假期
  • 總體統計、儀表板統計
  • 教學/幫助文件

提示：vAi 可以理解自然語言，您可以用日常對話的方式提問。

DNS 設定

這頁會一步步教你在網域註冊商的 DNS 管理介面新增 CNAME。 若你還未有網域，可先到 GoDaddy 或 Squarespace 購買。

步驟 1：找到 DNS 管理介面
  • 登入你的網域註冊商後台
  • 找到該網域（Domain）
  • 進入 DNS / DNS Records / 管理 DNS
步驟 2：新增 CNAME 記錄
新增一筆 CNAME
類型（Type）
CNAME
名稱（Host / Name）
www
TTL
Auto / 5m / 30m
指向（Value / Target / Points to）
{{ .CustomDomainCnameTarget }}
有些 DNS 介面要求「名稱」填完整網域，則填 www.你的網域（例如：www.example.com）。
步驟 3：回到系統檢查
  • 回到 自訂網域 & SSL
  • 依步驟按「檢查 DNS」直到顯示 OK
  • OK 後系統會在背景自動處理 SSL，你只需要在最後一步按「更新狀態」查看
常見問題／錯誤排除
  • 提示已存在記錄：如果 www 已有 A / AAAA / CNAME，請先刪除舊的再新增新的（同一個名稱不可同時存在衝突記錄）。
  • 找不到 CNAME：有些平台把它叫做「別名 / Alias」，但類型仍是 CNAME。
  • 檢查一直未完成：DNS 生效通常 5–60 分鐘，最慢可到 24 小時；也可能是你填錯了 Host/Target。
  • www 以外的網域：目前版本只支援 www。
