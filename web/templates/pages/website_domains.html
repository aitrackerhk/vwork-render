Cloudflare Domain Connect（自動）
  1. 客戶確保該網域已把 NS 指向 Cloudflare（DNS 在 Cloudflare 管）
  2. 客戶在你的平台填好 `www.customer.com` 並保存
  3. 客戶登入 Cloudflare → 選擇該網域 → 套用 Domain Connect 模板（會自動建立 CNAME 指向 {{.CustomDomainCnameTarget}}）
  4. 回來按「檢查 www DNS」看到 OK
注意：要真正做到「一鍵自動加 DNS」，你需要先把 Domain Connect 模板提交到官方模板庫，並按 Cloudflare 文件要求寄信給 `domain-connect@cloudflare.com` 讓模板上線。 在模板未上線前，自動模式會變成「指引式流程」。
