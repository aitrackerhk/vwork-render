快速入門指南

歡迎使用 vMarket！以下指南將幫助您快速了解 vMarket 的功能和使用方式。

什麼是 vMarket？

vMarket 是 V-sys 旗下的企業展示與線上市集平台。它讓企業的公司資訊、產品和服務在全球市場上獲得曝光，幫助買家輕鬆找到所需的產品和服務，同時讓賣家觸及更多潛在客戶。

vMarket 與 vWork 的關係

vMarket 與 vWork（V-sys 的企業管理系統）深度整合：

  • 資料來源 — 企業在 vWork 中管理公司資料、產品和服務
  • 自動同步 — vMarket 自動同步並公開展示 vWork 中的資料
  • 統一管理 — 在 vWork 更新一次，vMarket 同步呈現，無需重複維護
我是買家（訪客）

作為買家，您可以直接在 vMarket 上：

  1. 瀏覽所有已上架的 產品、服務 和 企業
  2. 使用 搜尋功能 快速找到您需要的產品或服務
  3. 使用 行業分類篩選 縮小瀏覽範圍
  4. 透過 地圖功能 查看企業的地理位置
  5. 點擊企業卡片進入企業的官方頁面了解更多詳情
我是賣家（企業）

作為賣家，您可以將公司、產品和服務展示在 vMarket 上：

  1. 首先在 vWork 中建立您的公司帳戶和資料
  2. 在 vWork 中新增您的產品和服務
  3. 前往 vMarket 設定 頁面，開啟「加入 vMarket」開關
  4. 您的公司、產品和服務將自動在 vMarket 上公開展示
完全免費 — 加入 vMarket 不收取任何費用，您可以隨時加入或退出。
下一步
  • 瀏覽市集 查看已上架的企業和產品
  • 了解 如何加入 vMarket
  • 查閱 常見問題
{{ else if eq .Category "browse" }}

瀏覽市集

vMarket 提供產品、服務和企業三大目錄，讓您輕鬆找到所需的資源。

瀏覽產品
  1. 進入 vMarket 首頁，點擊導航列中的 產品
  2. 頁面會顯示所有已上架企業的產品列表
  3. 產品按企業分組展示，每個企業顯示其公司名稱、Logo 和產品數量
  4. 點擊任意產品可進入該企業的產品詳情頁面
瀏覽服務
  1. 進入 vMarket 首頁，點擊導航列中的 服務
  2. 頁面會顯示所有已上架企業的服務列表
  3. 服務同樣按企業分組展示
  4. 點擊任意服務可進入該企業的服務詳情頁面
瀏覽企業目錄
  1. 進入 vMarket 首頁，點擊導航列中的 企業
  2. 頁面會顯示所有已加入 vMarket 的企業卡片
  3. 每張企業卡片包含：公司名稱、Logo、所屬行業、產品數量和服務數量
  4. 點擊企業卡片可進入該企業的官方頁面
行業分類篩選

在產品、服務和企業頁面中，左側（桌面版）或頂部（手機版）會顯示 行業分類側欄：

  • 點擊任意行業分類，頁面會自動篩選出該行業的企業、產品或服務
  • 點擊 全部分類 可取消篩選，顯示所有內容
  • 每個分類後方會顯示該分類下的企業數量
企業頁面導航

每張企業卡片提供多個快速導航按鈕：

  • 產品 — 查看該企業的所有產品
  • 服務 — 查看該企業的所有服務
  • 公司 — 查看企業的詳細資訊頁面
  • 地圖 — 在地圖上查看企業的地理位置
{{ else if eq .Category "search" }}

搜尋功能

vMarket 提供統一的關鍵字搜尋功能，讓您快速找到所需的產品、服務或企業。

使用搜尋
  1. 在 vMarket 首頁頂部可以看到 搜尋欄
  2. 輸入關鍵字，例如產品名稱、服務名稱或公司名稱
  3. 系統會即時搜尋並顯示匹配的結果
  4. 搜尋結果分為三類：產品、服務 和 企業
搜尋範圍

搜尋功能會同時在以下內容中查找匹配項：

類型搜尋欄位
產品產品名稱
服務服務名稱
企業公司名稱
搜尋結果
  • 搜尋結果會即時顯示，無需按下搜尋按鈕
  • 每次搜尋最多返回 20 筆 結果
  • 點擊搜尋結果中的任意項目，可進入對應的企業頁面查看詳情
搜尋提示
  • 使用具體的關鍵字可以獲得更精確的結果
  • 可以輸入部分名稱進行模糊搜尋
  • 如果搜尋結果太多，可以結合行業分類篩選進一步縮小範圍
{{ else if eq .Category "map" }}

地圖功能

vMarket 整合了 Google 地圖，讓您直觀地查看所有已上架企業的地理位置。

使用地圖
  1. 進入 vMarket 首頁，點擊導航列中的 地圖
  2. 地圖會顯示所有已加入 vMarket 且有提供地址的企業位置標記
  3. 使用地圖的縮放和拖拽功能瀏覽不同區域
查看企業資訊
  1. 點擊地圖上的 位置標記（圖釘圖示）
  2. 彈出的資訊窗口會顯示企業名稱和基本資訊
  3. 點擊資訊窗口中的連結，可以直接前往該企業的頁面查看更多詳情
地圖操作
  • 放大/縮小 — 使用滑鼠滾輪或地圖上的 +/- 按鈕
  • 拖拽移動 — 按住滑鼠左鍵拖拽地圖
  • 位置標記 — 每個標記代表一間企業的位置
企業的地址資訊來自其在 vWork 中設定的公司地址。如果您是企業用戶，請確保在 vWork 中正確填寫公司地址以在地圖上正確顯示。

加入 vMarket

了解如何將您的企業、產品和服務在 vMarket 上公開展示，觸及更多潛在客戶。

加入條件

要將您的企業展示在 vMarket 上，您需要：

  1. 擁有一個 vWork 帳戶（V-sys 的企業管理系統）
  2. 在 vWork 中建立您的 公司資料
  3. 在 vWork 中新增至少一項 產品 或 服務
加入步驟
  1. 登入 vWork 管理後台
  2. 在左側導航選單中找到 vMarket 設定（或在網站初始設定流程中選擇加入）
  3. 將「加入 vMarket」開關切換為 開啟
  4. 點擊 儲存
  5. 系統會自動將您的公司資料、產品和服務同步到 vMarket
完全免費 — 加入 vMarket 不收取任何費用。您可以隨時關閉曝光，資料僅顯示您已設定為公開的內容。
加入後的好處
免費曝光

您的公司和產品展示在 vMarket 公開頁面，讓更多潛在客戶找到您。

統一管理

在 vWork 更新一次，vMarket 同步呈現，無需重複維護資料。

信任提升

官方認證標誌讓客戶對您的企業更有信心。

業務拓展

與其他企業連結，互相擴大客戶群。

退出 vMarket

如果您不再希望在 vMarket 上展示您的企業，只需：

  1. 前往 vWork 管理後台的 vMarket 設定
  2. 將「加入 vMarket」開關切換為 關閉
  3. 點擊 儲存

退出後，您的公司、產品和服務將不再顯示在 vMarket 上。您隨時可以重新加入。

管理上架內容

了解如何控制在 vMarket 上展示的產品和服務內容。

自動上架機制

當您在 vWork 中加入 vMarket 後，以下內容會 自動 同步到 vMarket：

  • 公司資訊 — 公司名稱、Logo、行業類別、地址
  • 產品列表 — 所有設定為「在 vMarket 上顯示」的產品
  • 服務列表 — 所有設定為「在 vMarket 上顯示」的服務
控制產品/服務的顯示

每個產品和服務都有一個「在 vMarket 上顯示」的設定。加入 vMarket 後，新建立的產品和服務會 自動開啟 此設定。

如果您不希望某個產品或服務在 vMarket 上顯示：

  1. 前往 vWork 管理後台
  2. 打開該產品或服務的編輯頁面
  3. 找到「在 vMarket 上顯示」選項，將其關閉
  4. 儲存更改
上架條件

產品或服務要在 vMarket 上正確顯示，需滿足以下條件：

條件說明
企業已加入 vMarketvMarket 設定中的「加入 vMarket」開關為開啟狀態
企業狀態為啟用vWork 中的企業帳戶為正常啟用狀態
產品/服務狀態為啟用產品或服務未被停用或刪除
已開啟 vMarket 顯示產品或服務的「在 vMarket 上顯示」設定為開啟
更新展示資料

在 vWork 中所做的任何更新都會自動反映在 vMarket 上：

  • 修改產品名稱、描述、圖片 — vMarket 自動更新
  • 修改服務名稱、描述、圖片 — vMarket 自動更新
  • 修改公司名稱、Logo、地址 — vMarket 自動更新
  • 新增或刪除產品/服務 — vMarket 自動同步
預覽展示效果

您可以隨時前往 vMarket 頁面查看您的企業在市集上的展示效果。在 vWork 後台的 vMarket 設定頁面中，點擊「預覽 vMarket」按鈕即可快速前往。

常見問題

以下是 vMarket 用戶最常詢問的問題。

vMarket 是什麼？

vMarket 是 V-sys 旗下的企業展示與線上市集平台。它讓企業可以將公司資訊、產品和服務公開展示，幫助買家和賣家建立連結。

加入 vMarket 需要付費嗎？

完全免費。加入 vMarket 不收取任何費用。您只需要擁有一個 vWork 帳戶，即可在 vMarket 設定中一鍵加入。

我需要 vWork 帳戶才能在 vMarket 上展示嗎？

是的。vMarket 的資料來自 vWork 系統。您需要先在 vWork 中建立公司資料、產品和服務，然後透過 vMarket 設定將其公開展示。

作為買家，我需要帳戶才能瀏覽 vMarket 嗎？

不需要。任何人都可以直接瀏覽 vMarket 上的產品、服務和企業目錄，無需登入或註冊。

我可以控制哪些產品/服務在 vMarket 上顯示嗎？

可以。每個產品和服務都有一個「在 vMarket 上顯示」的設定。您可以在 vWork 中逐一控制哪些產品和服務要公開展示，哪些保持私有。

我可以隨時退出 vMarket 嗎？

可以。您只需要在 vWork 的 vMarket 設定中關閉「加入 vMarket」開關即可。退出後，您的所有資料將不再在 vMarket 上顯示。您隨時可以重新加入。

在 vWork 中更新資料後，vMarket 會自動更新嗎？

會。vMarket 與 vWork 自動同步。您在 vWork 中所做的任何修改（產品名稱、描述、圖片、公司資訊等）都會自動反映在 vMarket 上。

免費用戶和付費用戶有什麼區別？

免費用戶可以使用 vMarket 的所有基本功能（企業展示、產品上架等），但頁面上會顯示廣告。訂閱 vSuite 或 vSuite Pro 方案後，將享有無廣告體驗以及更多進階功能。

vMarket 上的企業地圖位置不正確怎麼辦？

企業的地圖位置是根據其在 vWork 中填寫的公司地址自動生成的。如果位置不正確，請：

  1. 前往 vWork 管理後台
  2. 修改公司設定中的 地址
  3. 確保地址填寫完整且準確
vMarket 支援哪些語言？

vMarket 支援三種語言：

  • 繁體中文（預設）
  • 简体中文
  • English

您可以透過頁面右上角的語言選擇器切換語言。
